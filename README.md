# Blind Assistance Glasses

An Arduino Uno based assistive device that helps detect obstacles using an ultrasonic sensor and alerts the user through a buzzer.

## Features

- Obstacle detection using HC-SR04 ultrasonic sensor
- Audible alert through buzzer
- Push-button ON/OFF toggle system
- Portable prototype for assistive applications

## Components Used

- Arduino Nano
- HC-SR04 Ultrasonic Sensor
- Active Buzzer
- Push Button
- Breadboard
- Jumper Wires

## Working Principle

1. User turns the system ON using the push button.
2. Ultrasonic sensor continuously measures distance.
3. When an obstacle is detected within the set threshold distance, the buzzer alerts the user.
4. Pressing the button again turns the system OFF.
